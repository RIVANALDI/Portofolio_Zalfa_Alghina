BabaShiritori Game

BabaShiritori Game adalah game edukasi yang dikembangkan oleh tim saya sebagai bagian dari inisiatif pengajaran di SMK Negeri 2 Malang dalam program P4. Terinspirasi dari permainan Jepang Babanuki dan Shiritori, game ini mengintegrasikan pembelajaran kosakata dengan gamifikasi untuk meningkatkan kemampuan bahasa Jepang siswa.
